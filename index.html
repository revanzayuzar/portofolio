Tentang Saya

Selamat datang di halaman website portofolio saya! Nama saya M. Revanza Yuzar, saya biasanya dipanggil Revan. Saya lahir di Sigli, kabupaten Pidie, provinsi Aceh pada tahun 2003 dan beragama Islam. Saya berasal dari kabupaten Pidie Jaya, provinsi Aceh, Indonesia. Saat ini, saya merupakan seorang mahasiswa di salah satu universitas swasta di provinsi Aceh, yaitu Universitas Jabal Ghafur. Saya berkuliah di jurusan Teknik Informatika karena saya mempunyai minat dan tekat yang besar dalam bidang Teknologi Informasi dan Digital. Saya juga memiliki ketertarikan terhadap Web Developer (Front-End). Saat ini, saya sedang mempelajari dan mencoba untuk mendalami tentang pemrograman web yaitu dengan menggunakan HTML, CSS, dan juga JavaScript. Dalam halaman ini, saya ingin membagikan sedikit tentang perjalanan dari pendidikan saya dan saya juga ingin memaparkan sedikit tentang karya website sederhana yang sedang saya kembangkan saat ini.

Saya percaya bahwa segala sesuatu dapat dicapai jika kita menjalaninya dengan penuh semangat dan bersungguh-sungguh. Hal itulah yang mendorong saya untuk terus berkembang dan memberikan yang terbaik dalam setiap project dan segala sesuatu yang saya kerjakan, serta berusaha untuk memberikan solusi yang kreatif dan juga efektif. Saya sering berimajinasi dan menuangkannya ke dalam berbagai hal yang saya lakukan, dan saya berusaha agar imajiansi tersebut dapat menjadi sebuah karya atau kenyataan.

Ketika saya tidak sibuk dengan aktivitas saya sebagai seorang mahasiswa, saya biasanya menghabiskan waktu luang dengan hobi-hobi saya, seperti mendesain sebuah website, mendengarkan musik, bermain piano, bermain game, dan melakukan editing-editing sederhana.

Terima kasih telah mengunjungi halaman website portofolio saya, dan saya berharap dapat berkenalan lebih lanjut dengan Anda dan berpotensi bekerja sama dalam project masa depan. Jangan ragu untuk menghubungi saya untuk pertanyaan atau peluang kerjasama!
